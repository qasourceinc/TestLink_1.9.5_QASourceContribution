Add Test case to Test Plan

Purpose:

This Page allows the user (with lead level permissions) to add test cases by either their components or their category levels into a Test plan.

To add Test cases:

  1. Click on a component to see all of its categories and all of its test cases. Clicking on a category will only show that categories test cases.
  2. Once you've selected the component/category you wish to draw test cases from use the radio buttons or check boxes to select the test cases you wish to import.
  3. When you are done click the 'Add Test Cases' button to import the test cases. Note: The system makes sure that the user does not import the same test case multiple times.
  4. You can also use filter according to Keywords.
