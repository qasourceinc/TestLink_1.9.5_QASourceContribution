General

Users can have different functionality available for browsing and editing. There are defined rights collections. Administrator can assign these rights only.
Default right is 'Guest'. This right allow to browse Test Specification and Test Results only (no edit). Anybody can obtain the right 'Guest' via login page.

Default rights description

Guest

This right allows to browse Test Specification, Keywords and Test Results and edit only personal data.

Tester

This right allows to browse Test Specification, Keywords and Test Results and edit Test execution.

Test Designer

This right allows to edit Test Specification, Keywords and Requirements.

Senior Tester

This right allows to edit Test Specification, Keywords, Requirements, Test execution and Create Builds.

Leader

This right allows to edit Test Specification, Keywords, Requirements, Test execution, Test Plan (include priorities, milestone and assignment) and Builds.

Admin

This right allows all. (Edit Products and Users in adddition.)
