General

Teslink allows shortcuts for a smart navigation. Use ALT + key.

List available shortcuts:

  • [h] Home
  • [i] User Info
  • [q] Logout/quit
  • [u] User administration (admin only)

Browser support

The accesskey attribute is currently supported by the following web browsers:

  • Internet Explorer 4 (Windows) 5+ (Mac)
  • Mozilla (Windows + Linux)
  • Netscape 6.1+ (Windows)
  • Opera 7 (Windows + Linux)

IE only selects the link. Enter key must follow.
