Assign Ownership and Priority

Purpose:

This feature allows the user to set the risk, importance, and owner of each test suite. The link 'Define Priority Rules' allows the user to set the definition of each risk/importance pairing. This definition is related just for active Test Plan.

Get Started:

  1. Click on a Test suite to see all of its nested Test Suites. Change the risk and importance using the radio buttons. The Test Suite owner can be set using the text box.
  2. Submit new values by click on 'Update' button.
