Print Test Specification

Purpose:

This functionality allows the user to print Test Cases by either their Test Suites or for a whole Test Project or Test Plan.

Get Started:

  1. Click on a required Test Suite, or Test cases to see all of the corresponding test cases below it. There will be also Test cases from all sub Test Suites.

  2. The 'Print option' table in navigation pane allows to define a range and format of printed data. See help for more information.

  3. Use your browsers print functionality to actually print the test cases.
    Note: Make sure to only print the right frame.
