Assign Testers to Test Plan

Purpose

This page allows users to define the list of approved users for each test plan. These users will able to modify data within the the Test Plan.

Get Started

  1. Choose to either manage rights by users or projects.
  2. Clicking on a user will allow you to select all of the Test Plan that the user is able to see.
  3. Selecting a test Plan allows you to select all of the users that are a part of that Test Plan.
